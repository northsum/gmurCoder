# gmurCoder

競プロ用リポジトリ

ディレクトリ構成は以下の通り

```
.
|
|- history ... 過去に解いた問題の記録
|   |- something...
|
|- playground ... 問題解答時の活動ディレクトリ。正味どこでやってもいい。
|   |- a.cpp
|
|- snipets ... 言語ごとのスニペットをまとめる。言語ディレクトリごとのsnipet.jsonを、vscodeのスニペットにコピペする。
|   |- c++
|   |   |- snipet.json
|   |   |- io.cpp
|   |   |- ...
|   |
|   |- python
|   |   |- snipet.json
|   |   |- ...
```
